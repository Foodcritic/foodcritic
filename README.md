# Food Critic

Food Critic is a lint tool for Chef cookbooks. It requires Ruby 1.9.3+.

# Homepage
[http://acrmp.github.com/foodcritic/](http://acrmp.github.com/foodcritic/)

# Building

    $ bundle install
    $ bundle exec rake

# Continuous Integration
